#### 7. axios 拦截器怎么配

```
    // 添加请求拦截器
    axios.interceptors.request.use(request => {
      // 在发送请求之前做些什么
      return request
    }, error => {
      return Promise.reject(error)
    })
    // 添加响应拦截器
    axios.interceptors.response.use(res => {
      // 对响应数据做点什么
      return res
    }, error => {
      return Promise.reject(error)
    })
```
